## 📥 安装步骤

### 使用安装包（推荐）

1. 从 [Releases](<repository-url>/releases) 下载最新安装包
2. 运行安装程序，按照提示完成安装
3. 启动应用

### 开发者安装

```bash
# 1. 克隆仓库
git clone <repository-url>
cd luuk

# 2. 安装依赖
npm install

# 3. 启动开发服务器
npm run dev
```

## 📁 添加图片库

### 首次添加库

1. 启动应用后，点击界面上的 **"管理"** 按钮
2. 选择 **"添加库"**
3. 浏览并选择包含图片的文件夹
4. 输入库名称（可选，默认使用文件夹名）
5. 点击 **"确定"**，应用将自动扫描图片

### 扫描进度

扫描过程中会显示进度条：
- 小库（<100 张）：几乎瞬间完成
- 中库（100-1000 张）：约 1-5 秒
- 大库（>1000 张）：约 5-60 秒

## 🎯 基本操作

### 浏览图片

- **网格视图**：显示所有图片的缩略图
- **瀑布流视图**：根据图片高度自适应布局
- **文件夹树**：按文件夹层级浏览

### 查看图片

点击任意缩略图进入查看器模式：
- **拖拽**：平移图片
- **滚轮**：缩放图片
- **双击**：适应窗口/实际大小切换

### 收藏管理

- 按 **`F`** 键：收藏/取消收藏当前图片
- 支持单图收藏和文件夹收藏
- 可切换单图模式/文件夹模式查看收藏

### 视图切换

- 按 **`F5`**：切换网格视图/查看器视图
- 按 **`F6`**：显示/隐藏文件夹侧边栏

## ⌨️ 快捷键

### 导航

| 快捷键 | 功能 |
|--------|------|
| `←` / `→` | 上一张 / 下一张 |
| `Home` / `End` | 第一张 / 最后一张 |

### 查看器控制

| 快捷键 | 功能 |
|--------|------|
| `0` | 适应窗口 |
| `1` | 实际大小 (100%) |
| `R` | 重置缩放/旋转/翻转 |
| `H` | 水平翻转 |
| `V` | 垂直翻转 |
| `I` | 显示图片信息 |

### 其他

| 快捷键 | 功能 |
|--------|------|
| `F` | 收藏/取消收藏 |
| `Space` | 幻灯片播放/暂停 |
| `Esc` | 关闭查看器 |
