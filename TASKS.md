## 📝 技术决策记录

### 2026-02-26

1. **框架选择**: Electron + React
   - 理由：生态成熟，开发效率高，便于后续集成 AI 功能

2. **构建工具**: Vite 7
   - 理由：开发速度快，配置简单，支持 HMR

3. **状态管理**: Zustand
   - 理由：轻量，API 简单，TypeScript 友好

4. **虚拟滚动**: @tanstack/react-virtual
   - 理由：高性能，支持动态尺寸，React 生态标准

5. **图片缩放**: react-zoom-pan-pinch
   - 理由：功能完整，支持触摸，性能优秀

6. **依赖清理**:
   - 移除未使用包：`adm-zip`, `electron-store`, `chokidar`, `sharp`, `better-sqlite3`, `qwen`
   - 移除冗余 scripts: `electron:dev`, `electron:build`, `lint`

7. **项目结构**:
   ```
   src/
   ├── components/     # React 组件
   │   ├── ImageViewer.tsx  (查看器 + 工具栏 + 信息面板)
   │   └── ImageGrid.tsx    (网格视图 + 虚拟滚动)
   ├── stores/         # Zustand 状态管理
   │   └── imageStore.ts
   ├── types/          # TypeScript 类型定义
   │   └── index.ts
   └── utils/          # 工具函数 (待补充)
   ```
